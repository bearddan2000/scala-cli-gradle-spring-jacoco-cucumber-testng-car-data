# scala-cli-gradle-spring-jacoco-cucumber-testng-car-data

## Description
A POC for spring app using testng,
jacoco and cucumber framework.

## Tech stack
- scala
- maven
  - spring
  - testng
  - jacoco
  - cucumber

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
